Changes to loss in step 3
=========================
change OKS to L1.
OKS - Object Keypoint Similarity - calculates an exponential similarity score between 0 and 1, factoring in the Euclidean distance, the bounding box scale (s), and a per-keypoint variance constant (σ). The loss is minimized by maximizing the similarity. It learns proportional, scale-invariant correctness. The gradient is dynamic. If a prediction is extremely far from the target relative to the σ tolerance, the exponential term evaluates to 0, the loss plateaus at 1.0, and the gradient vanishes (0). It only provides strong corrective gradients when the prediction enters the defined localized catchment area. So the loss is effective only when the prediction is close to the target. used for subjects with high deformation (human pose, hands, animals).
L1 loss - abs delta x+ abs delta y, the gradient (derivative) is +-1
